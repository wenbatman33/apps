# 音檔放置位置

把對應名稱的 mp3（或 ogg/wav，但路徑要改）放在這個資料夾，遊戲載入時會自動偵測並使用，沒有檔案時 fallback 到 WebAudio 即時合成的 placeholder 音。

| 檔名 | 用途 |
|------|------|
| `bgm.mp3` | 背景音樂（自動 loop） |
| `shoot.mp3` | 砲塔射擊 |
| `hit.mp3` | 子彈命中 |
| `enemy_die.mp3` | 怪物死亡 |
| `merge.mp3` | 砲塔合成升級 |
| `coin.mp3` | 撿金幣 / 購買 |
| `wave_clear.mp3` | 波次清除 |
| `click.mp3` | 按鈕點擊 |
| `upgrade.mp3` | 商店升級 |
| `game_over.mp3` | 遊戲結束 |
